catalog as a searchable browsing experience rather than one story per icon.

## Exports

- Hundreds of generated icon components, typically in Filled and Regular variants
- Sized icon variants for supported sizes

## Installation

```bash
pnpm add @axiscommunications/fluent-icons
```
