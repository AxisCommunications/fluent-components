<Subtitle>Contextual empty states for main, panel, submenu, and dialog layouts.</Subtitle>
<Description>
The empty view package provides presentation variants for no-data states.
Use the variant that matches the container where the empty state is rendered.
</Description>

## Exports

- MainEmptyView
- PanelEmptyView
- SubmenuEmptyView
- DialogEmptyView

## Installation

```bash
pnpm add @axiscommunications/fluent-empty-view
```

## Guidance

- MainEmptyView: use for full-page or large content areas.
- PanelEmptyView: use inside side panels and constrained surfaces.
- SubmenuEmptyView: use in compact navigation or submenu regions.
- DialogEmptyView: use in dialog content with limited height.
