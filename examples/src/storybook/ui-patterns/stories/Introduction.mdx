# Fluent UI Composite Components

Welcome to the **Fluent UI Composite Components** library. This project showcases 6+ custom composite components built strictly according to Fluent design guidelines and best practices.

## What's Inside

This Storybook contains interactive stories for:

-  **Form Patterns** вЂ” `FormField` for accessible form controls with validation
- **Data Display** вЂ” `StatCard`, `InfoCard`, `DataTable` for presenting information
- **Layouts** вЂ” `PageHeader` for consistent page structure

вњ“ **Fluent-First** вЂ” Only use `@fluentui/react-components` as building blocks

3. Check the **Accessibility** panel (a11y addon) for any violations
