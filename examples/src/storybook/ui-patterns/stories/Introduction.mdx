# Fluent Components

Welcome to the Axis Fluent component library for React and Fluent UI v9.
This Storybook documents reusable building blocks and usage patterns used
across applications.

## Packages In This Library

- `@axiscommunications/fluent-theme` - Axis branded Fluent themes
- `@axiscommunications/fluent-icons` - Axis icon set
- `@axiscommunications/fluent-hooks` - shared React hooks
- `@axiscommunications/fluent-styles` - shared style modules
- `@axiscommunications/fluent-stepper` - stepper and wizard flows
- `@axiscommunications/fluent-slider` - slider and range slider components
- `@axiscommunications/fluent-topbar` - app, org, and profile menu topbar
- `@axiscommunications/fluent-empty-view` - empty state views
- `@axiscommunications/fluent-password-input` - password input with reveal
- `@axiscommunications/fluent-illustrations` - Axis branded illustrations

## Usage Patterns

The usage patterns documented in this Storybook are developed by the
Design System Group and serve as global UX patterns.

Use them as the default approach when building new experiences to ensure:

- consistency across products
- predictable behavior for users
- accessibility and Fluent v9 alignment
- faster implementation with shared conventions

## Getting Started

1. Browse stories in the sidebar by package or pattern.
2. Use controls to explore variants and props.
3. Apply these patterns as-is unless your team has an approved exception.
