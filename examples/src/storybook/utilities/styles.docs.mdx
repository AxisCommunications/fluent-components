<Subtitle>Shared style helpers for Fluent UI tables, menus, tabs, navigation, and static app styling.</Subtitle>
<Description>
The styles package provides reusable class-producing hooks and static styles for
common application patterns built on top of Fluent UI components.
</Description>

## Exports

- useMainMenuContainerStyles
- useMainMenuTabListStyles
- useMainMenuTabStyles
- useNavigationStyles
- useRowStyles
- useColumnStyles
- useTableStyles
- useTabListStyles
- useTabStyles
- useScrollStaticStyles

## Installation

```bash
pnpm add @axiscommunications/fluent-styles
```

## Guidance

- Apply these hooks at the composition layer where Fluent primitives are assembled.
- Prefer shared styles for consistency across product surfaces rather than duplicating one-off overrides.
- Use static styles only once near the application root.
